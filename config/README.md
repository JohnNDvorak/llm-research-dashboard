# Configuration Files

YAML configuration files for the LLM Research Dashboard.

## Files (to be created in Step 3):

- `stages.yaml` - 8 pipeline stages with keywords
- `llm_config.yaml` - LLM provider settings, costs, selection rules
- `embedding_config.yaml` - Vector embedding provider settings
- `queries.yaml` - Search queries for arXiv/X/LinkedIn
- `budget_modes.yaml` - Cost optimization modes (cheap/balanced/quality)
